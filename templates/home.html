Home

Welkom op de website van de 'PICycle Speedometer'. Deze snelheidsmeter is mijn eindwerk voor de module project 1, een module die ik volg aan Howest binnen de opleiding NMCT. In feite is het is een soort integratie opdracht gebasseerd op de modules User 1, Device Programming 1, Datacommunication 1 en Datamanagement waarbij aan de hand van een Raspberry Pi data van een sensor ingelezen, opgeslaan in een database en gevisualiseerd wordt. Op deze 'homepagina' krijg je een kort overzicht te zien van de functionaliteiten van de website. Vergeet zeker niet eerst naar de pagina speedometer te gaan en daar jou gegevens te versturen.

Speedometer

Op deze pagina kan u uw persoonlijke gegevens meegeven. Deze zullen opgeslaan worden in onze database. Zeer belangrijk is om de diameter van uw wiel door te sturen. Indien u dit niet doet,z al de snelheidsmeter niet correct kunnen werken. Daarnaast kan u op deze pagina ook uw data visualiseren.
